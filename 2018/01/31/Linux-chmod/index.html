查看文件权限


文件权限

  • 读权限，表示你可以使用 cat 之类的命令来读取某个文件的内容；
  • 写权限，表示你可以编辑和修改某个文件；
  • 执行权限，通常指可以运行的二进制程序文件或者脚本文件，如同 Windows 上的 exe 后缀的文件，不过 Linux 上不是通过文件后缀名来区分文件的类型。

一个目录同时具有读权限和执行权限才可以打开并查看内部文件，而一个目录要有写权限才允许在其中创建其它文件，这是因为目录文件实际保存着该目录里面的文件的列表等信息。
